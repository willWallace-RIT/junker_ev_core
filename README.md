# Junker EV Framework

A modular standard for building electric vehicles from salvaged automotive components.

## Concept

Junker EV Framework defines a universal interface layer that allows:
- mixed-source electric motors
- second-life battery systems
- heterogeneous EV control units
- salvaged automotive electronics

to operate as a unified vehicle system.

## Goals

- Enable EV conversion using junkyard parts
- Standardize communication between incompatible hardware
- Reduce dependency on proprietary EV ecosystems
- Promote circular automotive manufacturing

## Core Components

- CAN Bus Message Standard
- Motor Interface Abstraction Layer
- Battery Module Standard
- Vehicle Control Unit (VCU) State Model

## System Philosophy

"If hardware is inconsistent, standardize the interpretation layer—not the components."

## Status

Early experimental specification.
